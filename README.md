# IONIC
Repositorio para la asignatura de acceso a datos.

## Primeros pasos con ionic
 * Instalar nodeJS 
 * npm install gradle
 * npm install cordoba -g
 * npm install native-run -g
 * npm install -g ionic
## Crear proyecto de ionic
 * ionic start myApp blank
## Run ionic app
 * cd myApp 
 * ionic serve
